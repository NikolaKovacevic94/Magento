Magento Community Edition Tutorial


Pred Vama je par video klipova koji će Vam pomoći da se snađete prilikom instalacije Magento Community Edition platforme. Pre svega, da se podsetimo koji su minimalni sistemski zahtevi:

  • Operativni sistem: Linux x86 - 64
  • Operativna (RAM) memorija: 2GB minimum
  • Veb server: Apache 2.x, Nginx 1.7.x
  • Baza podataka: MySQL 5.6 (Oracle ili Percona)
  • PHP: 5.4.x, 5.5.x, 7.0.x
  • SSL: Validan sigurnosni sertifikat koji je potreban za HTTPS
  • Server za elektronsku poštu: Mail Transfer Agent (MTA) ili SMTP server


Detaljnije video klipove možete pogledati na sledećem Youtube kanalu.
