Magento SOAP API


Magento SOAP (Simple Object Access Protocol) na grubom prevodu znači Pristupni protokol jednostavnih objekata. Ovaj API pruža mogućnosti da potpuno upravljate svojom elektronskom trgovinom; upravljajte klijentima, kategorijama, proizvodima, pa i prodajnim nalozima. Takođe, omogućava upravljanje i "potrošačkom korpom" i stanjem inventara. Magento SOAP podržava i XML-RPC, ali je podrazumevani protokol SOAP. Drugi Magentov API je REST.
